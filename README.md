# Player Management System

A full-stack CRUD web application built using:

Frontend: React  
Backend: Spring Boot  
Database: MySQL  

## Features
- Add, Update, Delete Players
- Search Players
- Dark / Light Mode Toggle
- Dashboard UI with charts
- Responsive design

## Tech Stack
React | Spring Boot | MySQL | REST API
